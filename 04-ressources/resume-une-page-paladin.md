## CHÂTIMENT DIVIN (SMITE)

```
APRÈS avoir touché → dépense 1 slot → +2d8 radiant

✓ CRIT (nat 20)? → SMITE TOUJOURS (dés doublés = 4d8)
✓ Boss ou ennemi important? → Smite
✓ Mort-vivant/Fiélon? → Smite (+3d8 au lieu de 2d8)
✗ Simple gobelin? → Garde ton slot
```

## LAY ON HANDS (Imposition des Mains)

```
Action: Touche quelqu'un → Soigne autant de HP que tu veux de ta pool

ASTUCE: Allié à 0 HP? → 1 point suffit pour le relever!
        Garde Soins pour les gros heals.
```

---

## SENTINEL (Mon Don)

```
✓ Tu frappes un ennemi → Sa vitesse = 0 (ce tour)
✓ Ennemi fait Disengage → Tu frappes QUAND MÊME
✓ Ennemi attaque un allié à côté de toi → RÉACTION: Tu frappes
```

---

## MON TOUR - CHECKLIST

```
□ MOUVEMENT (9m / 30ft)
□ ACTION (Attaque, Sort, Dash, etc.)
□ ACTION BONUS (si j'ai quelque chose)
□ RÉACTION (1 par round - Sentinel, Opp. Attack)
```
